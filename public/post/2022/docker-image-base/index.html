镜像源推荐中科大的源: https://mirrors.ustc.edu.cn

1. openjdk

官方镜像地址: https://hub.docker.com/_/openjdk

1.1. openjdk-8-jre-slim-buster

修改时区为中国上海； 修改为中科大的源；

1.2 openjdk-8-jre-slim-bullseye

修改时区为中国上海； 修改为中科大的源；

2. alpine

官方镜像地址: https://hub.docker.com/_/alpine

2.1. alpine-3.16-base

修改时区为中国上海； 修改为中科大的源；
